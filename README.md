# 📝 To-Do List Web Application

## 📌 Project Overview

This project is a **To-Do List Web Application** developed using **HTML, CSS, and JavaScript** as part of the **Oasis Infobyte (OIBSIP) Web Development Internship – Level 2 Task 3**.

The application helps users organize and manage their daily tasks efficiently. Users can add, edit, delete, and mark tasks as completed. The app separates pending and completed tasks, making task management simple and user-friendly.

---

## 🚀 Features

* Add New Tasks
* Edit Existing Tasks
* Delete Tasks
* Mark Tasks as Completed
* Separate Pending and Completed Task Lists
* Task Status Management
* Responsive User Interface
* Date & Time Tracking (Optional)

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
